中國八字排盤（WinForms / VB.NET / .NET Framework 4.7.2）
一款以 VB.NET + WinForms（.NET Framework 4.7.2） 開發的桌面應用程式，用於進行 中國八字（四柱）排盤。
介面簡潔、計算精準，適合研究者、教學者與一般使用者快速取得排盤結果。

📌 功能特色
🌓 輸入出生：年 / 月 / 日 / 時
🔢 自動計算：四柱八字（年柱、月柱、日柱、時柱）
📅 支援萬年曆轉換 & 閏月判定
⚙️ 以資料表方式計算天干地支
💾 可擴充模組：大運、流年、神煞、用神分析（視需要新增）

▶️ 使用方式
* 下載或 Clone 本專案：
  Shellgit clone https://github.com/n96831302-afk/BaZi-Chart
* 使用 Visual Studio 2026 開啟 .sln
* Build 專案後執行 WinForms 程式
* 輸入出生資訊 → 自動顯示八字

🔧 已知限制 / 待改進
* 大運、流年 為新式算法，跟傳統計算方法不同
* 部分曆法判定可改用天文庫強化精度
* UI 仍採基礎 WinForms 樣式，可進一步美化
